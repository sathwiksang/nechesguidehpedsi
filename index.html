Welcome to the Neches Cluster Connection Guide

This comprehensive guide will walk you through the process of connecting to the Neches cluster from a Windows computer. The Neches cluster is a high-performance computing resource available to University of Houston researchers and students. This documentation is designed to be clear and accessible for beginners, with step-by-step instructions and visual aids.

1
Connect to UH Secure Network via VPN

To access the Neches cluster, you must first be connected to the UH Secure network. If you're on campus, you may already be connected. For off-campus access, you'll need to use the Cisco VPN client.

Important: Download and install the Cisco Secure Client from the official UH IT website:
https://www.uh.edu/infotech/services/computing/networks/vpn/index.php

VPN Configuration:

When configuring the VPN, use one of the following host addresses:

  • VPN.uh.edu - Primary VPN service for campus remote access (recommended)
  • Full-VPN.uh.edu - Backup service for special cases or when primary VPN has issues
Cisco Secure Client VPN interface - Enter vpn.uh.edu and click Connect

Enter your CougarNet credentials when prompted and establish the VPN connection before proceeding.

2
Download MobaXterm

MobaXterm is a powerful terminal application for Windows that provides SSH client capabilities and many other useful features for remote computing.

Visit the official MobaXterm website to download the software:

https://mobaxterm.mobatek.net/download.html
MobaXterm download page - Choose Home Edition for free personal use

The Home Edition is free and sufficient for connecting to the Neches cluster. It includes all the necessary features for SSH connections and file transfers.

3
Select Installation Type

MobaXterm offers two versions: Portable and Installer. We recommend the Installer version for a more integrated Windows experience.

Choose between Portable and Installer versions - Installer is recommended
  • Installer Version (Recommended): Installs MobaXterm on your system with proper shortcuts and file associations
  • Portable Version: Runs without installation, useful for USB drives or restricted systems

After downloading, unzip the file if necessary and locate the installer executable.

4
Install MobaXterm

Run the MobaXterm installer and follow the on-screen installation wizard. The installation process is straightforward with default settings suitable for most users.

MobaXterm Setup Wizard - Click Next to proceed with installation

Accept the default installation directory and settings unless you have specific requirements. The installation typically takes less than a minute to complete.

5
Launch MobaXterm

After installation, launch MobaXterm from your Start Menu or desktop shortcut. You'll be greeted with the main interface showing various connection options.

MobaXterm main interface - Click on "Session" to create a new connection

The interface includes a file browser on the left, terminal area in the center, and various tools accessible from the top menu bar.

6
Configure SSH Session

Click on the "Session" button in the top-left corner to open the session configuration window.

Session settings window - Select SSH for secure connection

Follow these configuration steps:

  1. Click on SSH from the session types
  2. In the Remote host field, enter: neches.rcdc.uh.edu
  3. Check the "Specify username" checkbox
  4. Enter your CougarNet ID (the part before @cougarnet.uh.edu)

SSH session configuration with correct settings

After entering all details, click OK to initiate the connection.

7
Authenticate and Connect

Upon first connection, you may see a security prompt asking to accept the host key. Click Accept or Trust to continue.

Password authentication prompt - Enter your CougarNet password

Enter your CougarNet password when prompted. Note the following:

  • Password characters will not be visible as you type (this is normal security behavior)
  • Type carefully and press Enter when done
  • If you mistype, press Backspace to delete characters
Password Storage Option: After successful authentication, MobaXterm may offer to save your password for faster future connections. Only accept this option on your personal, secure computer. Never save passwords on shared or public computers.

Additional Information

Common Commands:

Once connected, you can use these common commands:

  • pwd - Show current directory
  • ls - List files and directories
  • cd - Change directory
  • module avail - List available software modules
  • module load [module_name] - Load a software module
  • squeue -u $USER - Check your job queue

File Transfer:

MobaXterm provides a built-in file browser on the left side of the interface. You can drag and drop files between your local computer and the cluster.

Troubleshooting:

  • Connection timeout: Ensure VPN is connected and active
  • Authentication failed: Verify your CougarNet credentials
  • Permission denied: Contact IT support to verify cluster access

Getting Help:

For additional assistance:
